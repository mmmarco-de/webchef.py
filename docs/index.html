Webchef is a simple, lightweight HTTP server, baked in Python. It's designed for easy preparation and serving, allowing you to quickly plate static files from your local station.

Signature Dishes (Key Features)

  • Quick Prep: Just fire up the Python script!
  • Clean Plates (URLs): Serves dishes without the .html garnish.
  • Custom Seasoning: Configure the server directly in the script and recipe file.
  • "Dish Not Found" Special (404 Page): Includes a dedicated "Dish Not Found" presentation (webchef.404.html).
  • Quiet Kitchen: Keeps console chatter to a minimum.

Recipe Adjustment (webchef.config)

The recipe file allows you to customize the server's ingredients and method: 


        {
            "kitchen_address": "127.0.0.1",  // The address where the Chef listens for orders (all interfaces)
            "oven_port": 8000,               // A different port for this custom setup
            "menu_directory": ".",           // Serve dishes from this specific folder
            "log_level": 30                  // Only show warnings and above (30 is logging.WARNING)
        }

Serving Instructions

  1. Procure the latest batch of webchef.py.
  2. Unpack it into your project's pantry.
  3. Adjust the recipe to your liking somewhere at the start of the file.
  4. Fire up the script: python webchef.py
  5. Your dishes are ready at http://127.0.0.1:8000/ (or the address and table number (port) specified in your recipe).

Enjoy plating your web creations with your friendly neighborhood Webchef!
